# Weekly Update Plan

Use this plan to keep the cookbook active and searchable.

## Monday

- Add one new recipe or improve one existing recipe.
- Update the README recipe list if needed.

## Wednesday

- Review issues and turn repeated questions into docs.
- Add one troubleshooting note.

## Friday

- Check GitHub traffic, stars, forks, and clicks to the website.
- Pick the next recipe based on search intent.

## Monthly

- Create a release tag.
- Summarize new examples.
- Refresh Topics and repository descriptions.
